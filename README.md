# Campus Notifications App

This is a simple React app built as part of the campus hiring evaluation.

## What this app does

- Gets access token using the given API
- Fetches notifications from backend
- Shows top 10 notifications
- Sorts notifications based on:
  - Placement > Result > Event
  - Latest first (timestamp)

## Features

- Basic UI with buttons
- Token-based API call
- Sorting logic implemented
- Simple filtering (optional if you added)

## Tech Used

- React (Vite)
- JavaScript
- Fetch API
